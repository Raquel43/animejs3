Aprendamos sobre Anime.js

Anime.js es una librería ligera basada en JavaScript. Se puede usar para animar diferentes propiedades CSS, SVG o atributos del DOM en una página web. Esta libreria permite controlar todos los aspectos de la animación y proporciona muchas maneras de especificar los elementos que se quieren emplear o las propiedades que se quieren animar.
